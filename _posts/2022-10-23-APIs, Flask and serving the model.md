# APIs are everywhere
<p>Not everywhere like your imaginary friend but they are everywhere in the tech world. Think of APIs as a way for your to open up your resources(data or methods) to other systems in a secure way.</p>
<br>
Examples? 

- Whenever you search something in Google, your searched text is passed from the front-end to the google search API. That API contacts the google server(backend) which does some computations and gives a response to the API which the Google webpage displays back to you</li>
- APIs don't have to be only in web applications. If you are familiar with pandas library in python then whenever you apply a pandas function to a dataset you are calling the pandas API</li>

# Why we need an API?
<p>We are going to host our model on a server. It's going to be a realtime prediction model i.e. it will always be ready to take an input of feature vectors and return a prediction. Like in the google example we will send our tweet for classification to a url which points to our ML API.</p>
